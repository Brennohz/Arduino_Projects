# Projetos Arduino

Este repositório contém códigos para projetos desenvolvidos com a plataforma Arduino.

## Projetos

### Jogo da Memória
Um jogo de memorização clássico. O Arduino sorteia uma sequência de LEDs e sons, e o jogador deve repeti-la pressionando os botões correspondentes na ordem correta.
* Demonstração em vídeo: [Vídeo no Google Drive](https://drive.google.com/file/d/1syI0QItPt37GWkArKCyD_P29qmIdhbsc/view?usp=sharing)

### Carro Bluetooth
Código para controle de um chassi de carrinho robótico com tração traseira através de comunicação Bluetooth. Utiliza o driver de ponte H L298N para controlar a velocidade e a direção dos motores de acordo com comandos enviados via aplicativo móvel.
* Demonstração em vídeo: [Vídeo no Google Drive](https://drive.google.com/file/d/1P1-joulibmRFrcQW6rK9lo0VsvfICjHs/view?usp=sharing)
